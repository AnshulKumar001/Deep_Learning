# Deep Learning

This repository contains deep learning concepts, implementations, and projects for learning and practice.

## About
This repo is created to understand deep learning from basic neural networks to advanced architectures. It includes practical implementations and real-world projects.

## Topics Covered

### Fundamentals
- Artificial Neural Networks (ANN)
- Perceptron & Multilayer Perceptron
- Activation Functions
- Forward & Backpropagation
- Loss Functions & Optimizers

### Deep Learning Architectures
- Convolutional Neural Networks (CNN)
- Recurrent Neural Networks (RNN)
- LSTM & GRU
- Transfer Learning

### Computer Vision
- Image Classification
- Image Preprocessing
- Feature Extraction

### Model Optimization
- Gradient Descent
- Overfitting & Regularization
- Dropout & Batch Normalization
- Hyperparameter Tuning

### Tools & Libraries
- Python
- NumPy & Pandas
- Matplotlib
- TensorFlow / Keras


## Installation

Clone the repository:

git clone https://github.com/AnshulKumar001/Deep_Learning.git

cd Deep_Learning

Install dependencies:

pip install -r requirements.txt

## Usage

Run Jupyter Notebook:

jupyter notebook

Run Python file:

python filename.py
